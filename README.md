이 프로젝트는 감정적으로 지치거나 우울한 순간에 음악을 통해 위로를 받을 수 있도록 기획된 개인 웹 프로젝트이다. 단순히 노래를 나열하는 페이지가 아니라, 사용자가 혼자 있는 시간에 조용히 감정을 정리하고 다시 일어설 수 있도록 돕는 공간을 만드는 것을 목표로 했다. “우울할 때 듣는 힐링송”이라는 주제 아래, 각기 다른 분위기와 메시지를 가진 곡들을 소개하고, 사용자가 직접 노래를 듣고 감상을 남길 수 있도록 구성하였다.

홈페이지는 HTML, CSS를 기반으로 제작되었다. HTML5를 사용해 페이지를 의미 있는 구조로 나누었으며, index.html을 중심으로 각 곡별 상세 페이지를 개별 HTML 파일로 구성하였다. CSS3를 활용해 그라디언트 배경, 카드형 UI, 반응형 레이아웃을 구현했고, 다크 모드와 라이트 모드에 따라 색상 변수가 변경되도록 설계하였다. JavaScript는 다크 모드 토글 기능과 LocalStorage를 이용한 사용자 설정 저장에 사용되었으며, 페이지를 이동해도 테마가 유지되도록 구현하였다. 음악 재생은 YouTube iframe embed 방식을 사용해 별도의 라이브러리 없이도 안정적인 동작을 가능하게 했다.

페이지 구성은 크게 메인 페이지와 트랙 페이지로 나뉜다. 메인 페이지(index.html)에서는 프로젝트의 주제를 소개하고, 추천 힐링송 3곡을 카드 형태로 배치하여 각 곡의 상세 페이지로 이동할 수 있도록 구성하였다. 각 트랙 페이지는 동일한 양식을 사용해 제목, 아티스트 정보, 영상 재생 영역, 곡 설명, 그리고 사용자가 감상을 기록할 수 있는 입력 영역을 포함한다. 모든 페이지 상단에는 메인으로 돌아갈 수 있는 링크와 다크 모드 전환 버튼을 배치해 사용성을 높였다.
