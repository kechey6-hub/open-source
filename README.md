이 프로젝트는 감정적으로 지치거나 마음이 내려앉는 순간에, 따뜻한 위로가 되어주는 음악을 추천하기 위해 만들어진 웹페이지입니다. 말로 표현하기 어려운 우울한 감정과 혼자 견뎌야 하는 상황에서, 음악 한 곡이 돌아오는 길을 밝혀주는 작은 빛이 될 수 있다는 생각에서 출발했습니다. 사용자는 메인 화면에서 3곡의 힐링 음악을 소개받고, 각 곡별 페이지에서 YouTube 영상과 간단한 소개 글을 확인할 수 있으며, 직접 감상 메모를 남길 수도 있습니다.

메인 페이지는 카드형 레이아웃과 그라디언트 스타일로 디자인되었으며, 시각적 피로를 줄이고 감성적인 분위기를 느낄 수 있도록 다크 모드와 라이트 모드를 지원합니다. 사용자가 선택한 테마는 LocalStorage에 저장되어 페이지를 다시 열어도 유지되도록 설계되었습니다. 각 곡 페이지에서는 제목, 아티스트 정보, 영상, 설명, 메모 입력 영역이 동일한 구성으로 제공되어 사용자 경험을 통일했습니다.

프로젝트는 HTML, CSS, JavaScript로 구현되었고, 오디오 재생은 YouTube embed 기능을 활용했습니다. 디자인 측면에서는 그라디언트 배경, Glassmorphism 카드 UI, 간결한 타이포그래피를 사용해 감성적인 분위기를 표현했습니다. 다크 모드 처리, 로컬 저장소 활용, UI 반응형 설계 등 기본 웹 기술에 대한 학습과 실습을 목표로 개발했으며, GitHub Pages를 통해 쉽게 배포할 수 있도록 구성했습니다.

이 웹사이트는 단순히 음악을 나열하는 페이지가 아니라, 사용자가 조용히 감정을 정리하고 위로를 받을 수 있는 작은 공간을 제공하고자 합니다. 앞으로는 더 많은 힐링 음악 카테고리, 감성 태그, 감상 공유 기능, AI 기반 음악 추천 등 다양한 방향으로 확장될 수 있으며, 누구든지 자유롭게 참여하고 발전시킬 수 있는 오픈 프로젝트입니다.

음악이 누군가의 하루를 조금이라도 견딜 수 있게 하는 힘이 되기를 바라며, 이 프로젝트가 작은 위로의 자리가 되기를 바랍니다.
